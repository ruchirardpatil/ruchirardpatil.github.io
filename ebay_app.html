• Project Title:
    Product Search Android App
  • Tech Stack:
    Java, Android Studio, Google Maps API, JSON, Node.js, eBay API, JavaScript

Project Details

  • Integrated the eBay API to allow users to perform dynamic searches based on their queries, fetching and displaying relevant product listings in real-time with detailed information such as pricing, seller ratings, shipping options, and availability.

  • Focused on creating a highly responsive and user-friendly interface, ensuring smooth navigation and accessibility across devices, while incorporating the Google Maps API to allow users to find nearby stores or products based on their geographical location, improving convenience and streamlining the shopping process.
