• Project Title:
    Website Syncing Ebay Data
  • Tech Stack:
    Angular, Node.js, TypeScript, JSON, MongoDB, eBay API, GCP

Project Details

  • Leveraged Angular to build an interactive and responsive front-end, enabling users to seamlessly browse, search, and interact with eBay product listings. Node.js was used for efficient backend development, while MongoDB stored essential user data. The application was deployed on GCP, ensuring scalability and high availability.

  • Utilized TypeScript to implement API calls to the eBay API, fetching real-time product and transaction data. The results were displayed in a structured JSON format, ensuring a fast and organized presentation of information tailored to individual users, enhancing the overall shopping experience.
